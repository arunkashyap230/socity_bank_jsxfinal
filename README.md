# Society Bank

A modern society banking management application built with React and Vite.

## Features

- Dashboard with key metrics
- Member management with KYC verification
- Account management
- Transaction processing (deposits, withdrawals, transfers)
- Loan management with guarantor tracking
- Reports and notices
- Summary views

## Tech Stack

- React 18
- Vite
- React Router
- Recharts
- Lucide React Icons

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
